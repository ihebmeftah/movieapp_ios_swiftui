# projectswift

A simple SwiftUI movie app demonstrating TMDB integration, user authentication, and favorites.

## Features

- Browse trending and upcoming movies
- View movie details
- Login / Register and manage profile
- Save favorites

## Requirements

- Xcode 15 or newer
- iOS 17 SDK (adjust as needed)

## Run

1. Open `projectswift.xcodeproj` in Xcode.
2. Select a simulator or device and run the app.
3. Ensure your `GoogleService-Info.plist` is present.
